# Workspace

This directory is for local working material that is not part of the stable MCP runtime.

## Layout

```text
workspace/
  experiments/  # Prototypes, one-off investigations, and temporary task work
  scratch/      # Disposable local files
```

Do not place production MCP code, reusable tools, or test reports here. Promote useful work into `tools/`, `crawler_core/`, `tests/`, or `docs/` once it becomes durable.
